## Process

1. Open a clean browser session or use each AI/search product's logged-out/default experience where possible.
2. Test the queries below in Google, ChatGPT with search, and Perplexity.
3. Record whether Hal is cited, which competitors or alternatives are cited, and which Hal page should be improved.
4. If a query has no Hal citation for two consecutive months, improve the mapped page with a clearer answer block, source links, table, or FAQ.
5. Do not add fake metrics, testimonials, benchmarks, or unsupported claims to chase citations.
